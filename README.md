# Telegram bot parses channel's name and tag

![Bot Preview](https://s12.gifyu.com/images/SWlS9.gif)

## How to Run

### Python

```
pip install -r requirements.txt
```

```
BOT_TOKEN=000:AAA python main.py
```

### Docker

```
docker build -t bot .
```

```
docker run -d -e BOT_TOKEN=000:AAA bot
```
